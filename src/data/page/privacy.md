# 隐私策略（Privacy Policy）

最后更新日期：2025年10月21日

我们非常重视访客的隐私。本网站不会收集、储存或使用任何访客的个人数据。

### 一、数据收集

本网站不使用任何形式的跟踪技术（包括但不限于 Cookie、分析工具或第三方统计脚本）。
访问本网站的过程中，服务器仅进行最基本的网络连接操作，不会记录或保存任何可识别个人身份的信息。

### 二、数据共享

由于我们不收集数据，因此不存在与任何第三方共享、出售或转让个人数据的行为。

### 三、外部链接

本网站可能包含指向外部网站的链接。我们不对这些外部网站的隐私政策负责。建议访客在访问这些网站前仔细阅读其隐私条款。

### 四、政策更新

如本政策有任何修改，我们将在本页更新修改日期。若继续访问本网站，即表示您已知悉并接受更新后的内容。
